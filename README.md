# About [BundesEcho](https://www.bundesecho.de)

BundesEcho is a (now open-source) archive for saving and searching political quotes, with a focus on (but not limited to) german politics.
You can find the website [here](https://www.bundesecho.de).

The aim of this project is to:
- hold public figures accountable for what they say
- give people the opportunity to inform themself about the  opinions in the political landscape
- provide sources and context to quotes to prevent misinformation and misrepresentation

# Features

## Extensive Search-Tool

The Search-Tool in combination with the Tagging-Systems of Quotes and Persons allows for detailed
searches about Parties and Topics, Regions and Timelines.
Because this can be overwhelming and not always needed, the standard search provides a simple
interface to filter via text.

## Submitting and Reviewing Quotes

To thrive from community-input, the structure allows anyone to contribute and the mods to review
these contributions.
The database is designed for this: Every Person and every Quote has
an internal version history to better manage these contributions.

## Tagging

As mentioned the tagging system allows for detailed searches. Tags can not only be attributed to
quotes but also to persons.

## Sharing

This project makes it easy to share projects: a short URL, copy-to-clipboard and even generating
images from the quotes can help improve to spread sources and awareness in political discussions.

# Versions

## 1.3.0 - PersonTagging, MultiLanguage and SecondarySources

- some bug fixes

### PersonTagging

Persons can now be tagged (with the same Tags). 
REGION and PARTY tags also show in the detailed view.

### MultiLanguage

Quotes in different languages or from different countries are now supported, if they also have a
german translation.
The original language is still visible in the detail-view for transparency.

### SecondarySources

Sometimes links stop working. To prevent this, archiving a website is key. The archiving link can
also be referenced in the creation of Quotes.
